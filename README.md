# 💰 Finance Dashboard

A simple and interactive finance dashboard built using React to help users track and understand their financial activity.

## 🔗 Live Demo

https://finance-dashboard-brown-nine.vercel.app/

## 🚀 Features

* Dashboard overview with Balance, Income, and Expense
* Transactions table with search functionality
* Role-based UI (Viewer / Admin)
* Insights section (Highest transaction)

## 🛠 Tech Stack

* React
* JavaScript
* Vite
* CSS (inline styling)

## 📌 How to Run Locally

```bash
npm install
npm run dev
```

## 📊 Overview

This project focuses on clean UI design, simple state management using React hooks, and intuitive user experience.
